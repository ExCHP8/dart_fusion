OnTile extension

Extension on ExpansionTile providing a convenient method to create a copy of the tile with modified properties.

Example usage:

ExpansionTile originalTile = // your original tile;
ExpansionTile modifiedTile = originalTile.copyWith(
  // modify properties here
);
